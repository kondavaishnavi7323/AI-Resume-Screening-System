# AI Resume Screening System

An AI-powered Resume Screening System developed using Python, Streamlit, and PyPDF2. The application analyzes uploaded resumes, extracts relevant skills, and generates a resume score based on predefined job requirements.

Overview

This project analyzes uploaded resumes and extracts skills using Python and Streamlit.

Features

- Resume PDF Upload
- Skill Detection
- Resume Scoring
- Candidate Evaluation

Technologies Used

- Python
- Streamlit
- PyPDF2

How to Run

pip install -r requirements.txt

python -m streamlit run app.py

Output

Displays detected skills and resume score.
